Customigurumi- Site Plan

Site Name

Customigurumi

This name represents a website where customers can order handmade crochet plush toys, customized to their preferences.

Site purpose

The website will allow users to:

  • Browse a catalog of available amigurumi designs with images and descriptions.
  • Customize their order by selecting size, colors, and additional details.
  • Place an order using an interactive form that saves data with localStorage.
  • Learn more about amigurumis, their creation process, and care tips.

Scenarios

  • How can I order a custom-made amigurumi
  • What are the available sizes and colors for customization?
